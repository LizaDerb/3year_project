Инструкция по использованию корпуса

Корпус содержит тексты школьных сочинений с сайта mogu-pisat.ru с частеречной разметкой.

Поиск

Корпус поддерживает следующие типы запросов:

  1. Поиск по лемме: введите слово без кавычек в любой форме: пример
  2. Поиск по словоформе: введите слово в двойных кавычках: "пример"
  3. Поиск по лемме и части речи: введите лемму и часть речи через + без пробела: говорить+PRTF
  4. Поиск по части речи: введите часть речи: VERB
  5. Поиск по биграммам и триграммам

Вы можете искать биграммы и триграммы любого типа поиска и комбинировать их. Для поиска би-/триграмм напишите запросы через пробел: "я" говорить+VERB

Вывод

Поиск выводит число результатов, предложение с результатом поиска, автора, название текста, ссылку на текст.

Тегсет

Частеречная разметка корпуса производилась с помощью морфологического анализатора pymorphy2, который имеет следующие обозначения для граммем, см. таблицу ниже. Именно они используются для поиска по частям речи.
